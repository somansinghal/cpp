# 🌟 About This Repository

Welcome to my **C++ Project Showcase**, a curated collection of **29 hands-on implementations** built while exploring the fundamentals and advanced concepts of C++.

This repository documents my journey from writing simple programs to implementing advanced Object-Oriented Programming concepts such as inheritance, abstraction, polymorphism, operator overloading, dynamic memory management, and STL.

Instead of treating them as traditional lab exercises, I have organized each implementation as an individual project to create a more structured and practical learning experience.

This repository is designed to strengthen problem-solving skills, build a solid OOP foundation, and demonstrate real-world applications of core C++ concepts.

### 🧱 Object-Oriented Programming (OOP)

### 🏗️ Constructors & Memory Management

* Parameterized Constructors

# 🚀 How To Run

### Clone Repository

```bash
git clone https://github.com/somansinghal/cpp.git
```

```bash
cd cpp
```

### Compile


### Run

```bash
./output
```
