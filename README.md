# 🚀 How To Run

### Clone Repository

```bash
git clone https://github.com/somansinghal/cpp.git
```

### Open Folder

```bash
cd cpp
```

### Compile

```bash
g++ filename.cpp -o output
```

### Run

```bash
./output
```
